# UnBound
NP-Hard Problem Solver &amp; Approximation Analyzer

NP-Hard Problem Solver & Approximation Analyzer

Real-World Context
Many real-world problems (routing, scheduling, resource allocation) are computationally hard (NP-hard). Exact solutions are often infeasible for large inputs.

Problem Statement
Design a system that:
Solves NP-hard problems using exact algorithms (for small inputs).
Applies approximation algorithms for large inputs.
Compares optimal vs approximate solutions.

Objectives
Understand computational hardness.
Implement exact and approximation techniques.
Analyze trade-offs (optimality vs efficiency).
